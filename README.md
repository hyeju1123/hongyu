# 👋🏼 소개

<br />

<div  align="center">
<h3>암기중 - HSK 중국어 단어 학습 어플리케이션</h3>
때는 HSK(중국어계의 토익) 공부를 깔짝이던 시절.. <br />
외워도 끝이 없는 <b>한자를 익히는 데 도움이 된 두가지</b>가 있으니! <br /><br />
<b>1 - 한자 부수(亻, 冫, 艹 ... 이런 거) 의미 공부하기 🔖<br /></b>
<b>2 - 직접 손으로 써보기 ✍🏻<br /><br /></b>
하다보니 <b>HSK 단어, 부수 정리, 다양한 암기 시험(필기 포함)</b> 기능<br />
이 모두 들어간 <b>HSK 어플</b>을 찾다가 없어서 직접 만든 프로젝트
</div>
